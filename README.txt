1. Скопировать файлы app.py, train.py и requirements.txt в необходимую директорию.

2. Установить необходимые пакеты указанные в файле с зависимостями requirements.txt (в CMD ввести команду): 
pip install -r requirements.txt

3. Запустить app.py из директории, в которой расположен этот файл (в CMD ввести команды):
cd "file_path"
python app.py

или (для запуска в отдельном окне)

cd "file_path"
start /MIN "" python app.py

4. Передать строку запроса через браузер в форме http://127.0.0.1:5000/get_prediction?arg1=value1&arg2=value2...argN=valueN
, где arg1, arg2,...,argN - имена параметров (p01, p02, ..., p10 согласно заданию);
	value1, value2, ..., valueN - числовые значения параметров;
	local_host_IP = http://127.0.0.1:5000 - задаётся в случае когда программа-сервер, работает на том же компьютере, что и программа-клиент.
Для тестирования запрос можно сгенерировать в Postman, установив метод POST и заполнив форму запроса (соответствующие поля с названиями и значениями параметров). 

PS. При передаче запроса выполняется проверка наличия моделей в папке с файлом app.py. Если моделей нет, выполняется обучение моделей и их сохранение в папку '\models'. 
При отсутствии '\models' папка создается автоматически.
